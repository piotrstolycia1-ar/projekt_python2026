#projekt_python2026

Mini projekt SCADA wykonany w Pythonie z wykorzystaniem biblioteki PyQt5.

Opis projektu:
Aplikacja przedstawia wizualizację instalacji technologicznej złożonej z czterech zbiorników,
rurociągów oraz elementów dynamicznych (pompa, grzałka).
Proces przebiega automatycznie, a użytkownik ma możliwość zadawania parametrów początkowych.

Projekt zawiera:
- wizualizację poziomów i temperatur,
- ekran raportów i alarmów,
- logikę procesu oddzieloną od warstwy wizualnej,
- testy jednostkowe wykonane w bibliotece pytest.

Technologie:
- Python
- PyQt5
- pytest
- GitHub
